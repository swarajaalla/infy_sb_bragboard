# Infy_SB_Bragboard
Bragboard is an internal recognition tool for workplaces that lets employees appreciate their colleagues by posting shot-outs.
It Promotes a culture of appreciation by allowing tagging,commenting, and company-wide visibility.Admins can monitor engagement and moderate flagged content.

-------------------------------------------
Techstack : 
Frontend : React.js + Tailwind CSS
Backend : FastAPI
Database : PostgreSQL
Authentication : JWT (access+Refresh)
---------------------------------------------

Modules :
A > User Management & Authentication
B > Shot-out creation & Feed
C > Tagging reactions & Comments
D > Admin Tools & Analytics
